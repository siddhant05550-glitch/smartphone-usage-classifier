# smartphone-usage-classifier
This project focuses on building a Machine Learning-based classification system to detect smartphone usage patterns and predict potential overheating risks before they occur.
The model analyzes multiple sensor inputs such as CPU usage, screen time, battery drain rate, touch frequency, motion data, and ambient temperature to classify device behavior into:
Normal Usage (0)
Heavy Usage / Overheating Risk (1)
Unlike traditional threshold-based systems, this approach uses data-driven modeling to identify complex usage patterns that may lead to overheating.
Key features of the project include:
End-to-end Machine Learning pipeline including EDA, preprocessing, training, and evaluation
Implementation of multiple models such as Logistic Regression, KNN, and SVM
Feature selection using PCA and statistical methods
Performance evaluation using accuracy, precision, recall, F1-score, and ROC-AUC
Handling of class imbalance using SMOTE
This system can be integrated into smartphones to enable early overheating detection, automatic performance optimization, improved battery life, and enhanced device safety.
